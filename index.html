关于本站

广州地铁报站模拟器

广州地铁报站模拟器网页版（Guangzhou Metro Announcement Simulator Web）是一款由地铁爱好者为广大同好打造的纯前端交互项目。 我们致力于 100% 还原广州地铁列车的车内广播与 LED 走字屏。

本模拟器支持广州地铁1号线、2号线、3号线、4号线、5号线等各大主力线路的报站模拟。 无论您是想体验经典列车的“下一站是...（The next station is...）”粤普英三语广播，还是想模拟车门上方 LED 屏幕的滚动字幕、换乘路线提示，都可以在本站一键体验。

相比传统的 Windows 客户端，本站采用纯 HTML5 / CSS3 / JavaScript 技术开发，完全免下载，无病毒误报，完美自适应手机和电脑端，随时随地在浏览器中即可畅玩。
